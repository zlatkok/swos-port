Function keys when inputing text in menus:

    enter       - accept input and return
    escape      - abort and discard input
    left shift  - move cursor one position to the left
    left arrow
    right shift - move cursor one position to the right
    right arrow
    home        - move cursor to the beginning
    down arrow
    end         - move cursor the the end
    up arrow
    backspace   - delete a character left from the cursor
    delete      - delete a character right from the cursor
